# Telecom Customer Churn Analysis

## Project Overview
Analysis of customer churn for a California-based telecom company using SQL and Power BI.
The goal is to identify why customers leave, which segments are at highest risk, and the revenue impact.

## Dataset
- Source: Maven Analytics
- 7,043 customers | 38 columns
- Covers demographics, services, contract info, and churn status

## Tools Used
- **MySQL** — Data exploration, cleaning, and analysis
- **Power BI** — Interactive dashboard

## Key Findings
- Overall churn rate: **28.4%**
- Revenue lost to churn: **$3.7M**
- Highest risk segment: Month-to-Month contracts (51.7% churn rate)
- Fiber Optic customers churn at **42.1%** vs 20% for DSL
- 77% of customers who churn do so within the first 6 months
- Top churn reason: Competitor had better devices/offer (45% of churned customers)
